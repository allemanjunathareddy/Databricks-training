# Day 3 - SQL Joins Assignment

## 📌 Overview

Day 3 focused on understanding SQL JOIN operations and how data from multiple tables can be combined using different join techniques.

This assignment was based on an online learning platform database containing:

- students
- courses
- enrollments
- instructors

The assignment mainly focused on solving real-world SQL problems using different types of JOIN operations.

---

## 📚 Topics Covered

### SQL JOIN Operations
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN logic
- CROSS JOIN

### Assignment Tasks
- Students with enrolled courses
- Courses without enrollments
- Instructors with assigned courses
- Courses without instructors
- Students without enrollments
- Full student enrollment report
- Instructor-course report
- Student, Course, and Instructor report
- CROSS JOIN practice queries

- 1 Bonus CROSS JOIN Query
