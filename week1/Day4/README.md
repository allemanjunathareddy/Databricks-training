# Day 4 - Window Functions, CTEs & Advanced SQL Analytics

## 📌 Overview

Day 4 focused on advanced SQL analytical concepts using Window Functions and Common Table Expressions (CTEs).

This assignment was based on a retail sales dataset containing:

- employees
- customers
- orders

The main objective was to understand how advanced SQL is used in real-world business reporting, ranking, cumulative calculations, and analytical problem solving.

This level of SQL helps in transforming raw data into meaningful business insights.
