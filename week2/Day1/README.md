# Day 1 - NULL Functions in MySQL

## 📌 Overview

Day 1 of Week 2 focused on handling NULL values in SQL using different MySQL NULL functions and conditions.

The assignment included practical exercises on NULL replacement, NULL-safe calculations, filtering missing data, and handling incomplete records in real-world scenarios.

The main objective was to understand how NULL values affect SQL operations and how to manage them effectively for reliable query results.
